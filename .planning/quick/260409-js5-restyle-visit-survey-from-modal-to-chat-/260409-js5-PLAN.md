- `hidden` — initial, nothing rendered
- `bubble` — circular chat icon visible at bottom-right (brief state, auto-transitions to `open`)
- `open` — chat window expanded showing message + options
- `thankyou` — shows follow-up message, auto-closes after 2 seconds

- Fixed position: `bottom-6 right-6` (above the Prometheus watermark at `bottom-3 right-3`)

- Auto-transition to `open` state after a brief 600ms pause so user sees the bubble arrive

**Chat window (when open):**
- Positioned `fixed bottom-6 right-6`, width `w-80` (320px), max-height reasonable
- Border: `border border-[var(--border)]`, bg: `bg-[var(--bg)]`, shadow: `shadow-lg`
- Editorial styling: no rounded corners (or minimal `rounded-sm` at most), clean lines

- **Header bar:** Small bar at top with text "Monty" (or empty) and an X close button (right-aligned). Close button sets `sessionStorage.setItem('visit-survey-done', 'true')` and transitions to `hidden`.

- **Message area:**
  - Row with circular avatar (left) + message bubble (right)
  - Avatar: `next/image` with `/monty-pixel-body.png`, 40x40, `rounded-full object-cover` with `object-[center_15%]` to crop to the face area
  - Message bubble: Light background offset (`bg-[var(--bg)] border border-[var(--border)]`), contains text: "Thanks for checking out my site! What brought you here?"
  - Animate in with `motion.div`: `initial={{ opacity: 0, y: 10 }}` `animate={{ opacity: 1, y: 0 }}` with a slight delay

- **Quick-reply buttons:**
  - Render below the message as horizontal-wrapping pill buttons: `flex flex-wrap gap-2 mt-3`
  - Each pill: `px-3 py-1.5 text-xs font-medium border border-[var(--border)] bg-[var(--bg)] hover:opacity-60 transition-opacity cursor-pointer`
  - Keep the same OPTIONS array with same values
  - Each button MUST have `data-umami-event="visit-reason"` and `data-umami-event-reason={opt.value}` (identical to current)

- **Thank-you state:**
  - Replace message + buttons with new message from Monty: "Awesome, welcome! Enjoy exploring."
  - Same avatar + bubble layout
  - After 2 seconds (`setTimeout`), transition to `hidden`

**Animation details (motion/react):**
- Use `AnimatePresence` to wrap the chat window for enter/exit animations
- Chat window enter: `initial={{ opacity: 0, y: 20, scale: 0.95 }}` `animate={{ opacity: 1, y: 0, scale: 1 }}` `exit={{ opacity: 0, y: 20, scale: 0.95 }}`

- Bubble enter: `initial={{ y: 100, opacity: 0 }}` `animate={{ y: 0, opacity: 1 }}` with `{ type: "spring", stiffness: 300, damping: 25 }`
- Check `useReducedMotion()` from motion/react — if true, skip animations (set all initial values to their final state)
