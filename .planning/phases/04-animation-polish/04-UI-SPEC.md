## Animation Contract

This section is specific to Phase 4 and documents the interaction states and motion values all downstream consumers must implement exactly.

### Page Transitions (DSGN-03, D-01, D-02)

| State | opacity | y | Duration | Easing |
|-------|---------|---|----------|--------|
| Exit | 1 → 0 | 0 → 10px | 200ms | ease-out |
| Enter | 0 → 1 | 20px → 0 | 300ms | ease-out |
| Total budget | — | — | ~400ms max | — |

**Reduced motion override (D-10):** Exit: `opacity 1→0`, `duration: 150ms`, no y-axis movement. Enter: `opacity 0→1`, `duration: 150ms`. Skip y transforms entirely.

**Implementation target:** `src/app/template.tsx` — upgrade existing opacity-only fade.

### Scroll Reveal — All Pages (DSGN-04, D-05, D-06)

| Property | Value |
|----------|-------|
| Initial state | `opacity: 0`, `y: 60px`, `scale: 0.95` |
| Animated state | `opacity: 1`, `y: 0`, `scale: 1` |
| Trigger | 30% of element inside viewport (`amount: 0.3`) |
| Duration | 600ms |
| Easing | `easeOut` |
| Play behavior | Once only — `viewport={{ once: true }}` |
| Child stagger | 0.15s delay per child |

**Reduced motion override (D-10):** Render children directly with no motion wrapper. Elements appear instantly with no transform.

**Implementation target:** `src/components/animations/scroll-reveal.tsx` (new component).

### Homepage Parallax (HOME-03, D-04)

Three distinct parallax speeds for the homepage only:

| Layer | Speed Multiplier | Effect |
|-------|-----------------|--------|
| Background decorative | `0.2` | Subtle drift, barely perceptible |
| Mid-layer content blocks | `0.5` | Medium vertical shift |
| Text with rotation | `0.5` + rotate `[-2, 2]` degrees | Subtle tilt on scroll |

**Implementation:** Motion `useScroll` + `useTransform` via `ParallaxLayer` wrapper. GSAP ScrollTrigger for complex staggered homepage hero sequences.

**Reduced motion override (D-10):** `ParallaxLayer` renders static — `y: 0`, `rotate: 0`. No Lenis smooth scroll either.

### Homepage GSAP Cinematic Entry (HOME-03, D-03)

| Property | Value |
|----------|-------|
| Initial | `opacity: 0`, `y: 60px`, `scale: 0.95` |
| Animated | `opacity: 1`, `y: 0`, `scale: 1` |
| Stagger | `0.15s` between `.hero-element` children |
| Duration | `0.8s` per element |
| Easing | `power2.out` (GSAP) |
| ScrollTrigger start | `"top 70%"` (= 30% into viewport) |
| Plays | Once (`once: true`) |

**Reduced motion override (D-10):** Skip GSAP initialization entirely. Elements render at full opacity with no transform.

### Project Card Hover Reveal (PORT-02, D-08, D-09)

**Desktop hover state:**

| Property | Value |
|----------|-------|
| Overlay background | `rgba(0, 0, 0, 0.75)` over the card image area |
| Overlay enter | slides up from bottom: `y: 100% → 0` |
| Overlay duration | 250ms |
| Overlay easing | ease-out |
| CTA label | "View Project →" |
| CTA font size | 14px, weight 600 |
| CTA color | `var(--accent)` (`#2563eb` / `#3b82f6`) |
| Tags shown | All project tags rendered inside overlay |
| Tag styling | Existing pill style (12px, neutral-100/800 background) |

**Mobile tap fallback (D-09):**
- First tap: overlay becomes visible (same y-axis slide animation at 250ms)
- Second tap or tap-outside: overlay dismissed (reverse slide, 200ms)
- Touch target for the overlay CTA: 44px height minimum

**Reduced motion override (D-10):** Hover shows overlay instantly (no slide). Tap toggles overlay instantly. No duration, no transform.

**Implementation target:** `src/components/projects/project-card.tsx` — replace `hover:border-accent/50` CSS with Motion `whileHover` + `useState` tap fallback.

### Lenis + GSAP Sync (HOME-04, D-07)

| Constraint | Value |
|------------|-------|
| Lenis lerp | 0.1 (existing, keep) |
| GSAP lag smoothing | `0` (prevents scroll jump on tab focus) |
| GSAP time → Lenis | multiply by 1000 (seconds → ms) |
| Plugin registration | `gsap.registerPlugin(ScrollTrigger)` at module top |
| Reduced motion | Skip Lenis initialization entirely — native scroll returns |

**Implementation target:** `src/components/providers/lenis-provider.tsx` — add GSAP ticker sync.

## Accessibility Contract

| Requirement | Implementation |
|-------------|----------------|
| `prefers-reduced-motion` | `useReducedMotion()` from `motion/react` on all Motion components; explicit check before GSAP and Lenis init |
| Mobile tap-reveal | Overlay dismisses on tap-outside — no keyboard trap |
| Project card CTA | "View Project →" link is keyboard focusable even without hover |
| Minimum touch target | 44px height on overlay CTA and any tappable overlay area |
| Animation property constraint | Only `transform` and `opacity` animated — never `width`, `height`, `padding`, `margin` |
