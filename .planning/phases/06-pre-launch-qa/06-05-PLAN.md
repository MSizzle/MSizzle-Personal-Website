**Category A — Live grep gate (scriptable):**

```bash
EVID=.planning/phases/06-pre-launch-qa/evidence/seo/grep-live.txt
> "$EVID"

# Sample pages
SLUG=$(curl -s https://montysinger.com/blog | grep -oE 'href="/blog/[a-z0-9-]+' | head -1 | sed 's|href="/blog/||')
for ROUTE in / /about /prometheus /newsletter /uses "/blog/$SLUG"; do
  echo "=== $ROUTE ===" >> "$EVID"
  HTML=$(curl -sL "https://montysinger.com$ROUTE")

  for PATTERN_LABEL in 'em-dash:—' 'mdash-entity:&mdash;' 'nyc:\bNYC\b' 'new-york:\bNew York\b' 'based-in:\bbased in\b' 'investor:\binvestor\b' 'investing:\binvesting\b' 'leerink:\bLeerink\b' 'legacy-domain:msizzle.com'; do
    LABEL="${PATTERN_LABEL%%:*}"
    PAT="${PATTERN_LABEL#*:}"
    COUNT=$(echo "$HTML" | grep -cE "$PAT" || echo 0)
    echo "  $LABEL: $COUNT" >> "$EVID"
  done
done

# Any non-zero count = FAIL
grep -v ": 0$" "$EVID" | grep -v "^===" | grep ":" && echo "GREP FAIL" || echo "GREP PASS"
```

Any line showing a non-zero count in a non-header, non-`PASS` line = blocking fail. Open `06-0X-FIX-live-copy-<pattern>.md` for each pattern that hit, listing the offending route.
