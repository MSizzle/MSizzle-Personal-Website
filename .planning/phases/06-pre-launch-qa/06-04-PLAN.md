2. **Fetch the live post HTML:**
   ```bash
   SLUG=<pick-from-step-1>
   curl -sL "https://montysinger.com/blog/$SLUG" > .planning/phases/06-pre-launch-qa/evidence/notion-longtail/post.html
   ```
3. **Enumerate Notion image URLs.** The project uses an image-proxy pattern (per STATE decisions: "Image proxy route chosen over build-time download"). Images in the rendered HTML will either be:

   - served from `amazonaws.com` / Notion's `file.notion.so` (signed URLs, time-limited)
   ```bash
   grep -oE 'src="[^"]+"' .planning/phases/06-pre-launch-qa/evidence/notion-longtail/post.html \
     | grep -oE 'https?://[^"]+' | sort -u > .planning/phases/06-pre-launch-qa/evidence/notion-longtail/image-urls.txt
   ```
4. **Status-check each image URL:**
   ```bash
   > .planning/phases/06-pre-launch-qa/evidence/notion-longtail/image-status.txt
   while read url; do
     CODE=$(curl -sI -o /dev/null -w "%{http_code}" "$url")
     echo "$CODE $url" >> .planning/phases/06-pre-launch-qa/evidence/notion-longtail/image-status.txt
   done < .planning/phases/06-pre-launch-qa/evidence/notion-longtail/image-urls.txt
   ```
